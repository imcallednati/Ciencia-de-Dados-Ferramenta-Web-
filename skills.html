HABILIDADES

PYTHON

Python é uma linguagem de programação de alto nível, de código aberto que é amplamente utilizada em diversas áreas como Ciência de dados, Machine learning (ML), Automação. É considerada uma linguagem amigável para iniciantes, com sintaxe fácil de ler e que exige poucas linhas de código.

HTML

O HTML é o componente básico da web e é usado para criar páginas e aplicações online. Ele é composto por uma série de marcações, chamadas tags, que indicam aos servidores da web como o conteúdo deve ser apresentado no navegador.

CSS

CSS é uma linguagem de programação que define como os dados são apresentados no navegador.
